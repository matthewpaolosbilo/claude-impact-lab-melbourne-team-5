00 — Pixel design language
The Pixel Principle
Three rules govern every component: hard edges (no border-radius), 2px outlines instead of box borders, solid offset shadows instead of blur. Together they create that handcrafted pixel-art feel.
Hard edges — no radius
No border-radius on pixel components. Crisp 2px outlines only. Rounding is reserved for non-pixel surfaces (pills, tags).
Offset shadow — not blur
Solid 3px 3px 0 offset. Ink-coloured or accent-coloured. Never box-shadow with blur radius.
8px pixel grid
All spacing and sizing snaps to an 8px base unit. Spacing tokens: 4, 8, 12, 16, 24, 32, 48, 64px.

06 — Icon language
Emoji + Pixel Icons
Native emoji for personality and social context. Streamline Pixel icons (streamlinehq.com/icons/pixel) for UI chrome. Never mix both in the same component.

Usage rules
✅ Bungee + emoji in headings → "Music nerds 🎧"
✅ Streamline Pixel for nav, settings, actions
✅ 1 emoji max per badge or button label
✅ Emoji in interest tags, always paired with text
❌ Never mix emoji + pixel icon in the same component
❌ Emoji solo — always pair with readable text in UI

09 — Voice & tone
Copy Guidelines
Write like a person who actually uses the app. Warm, punchy, direct. Never corporate. Community copy celebrates belonging and makes people feel seen.
✅ do
"You joined Pixel Art Club — 892 people are hyped to meet you 🎉"
❌ don't
"You have successfully joined the community. Welcome aboard."
✅ do
"Nobody in your area yet — be the first to plant a flag 🚩"
❌ don't
"No communities found in your location."
✅ do
"3 people nearby are also into hyperpop ⚡ say hi?"
❌ don't
"3 users with matching interests have been found in your area."
✅ do
"Something broke — not the vibe. We're on it ⚡"
❌ don't
"An error has occurred. Please try again later or contact support."
10 — Design tokens · engineer handoff
CSS Tokens
Copy the block below directly into your project's token or variables file. Includes the Google Fonts import, all colour tokens, pixel system, and motion curves.
/* ============================================================
   PXL // DS — Design Tokens v3.0
   Copy into: tokens.css / variables.css / :root stylesheet
   ============================================================ */

/* Google Fonts import */
@import url('https://fonts.googleapis.com/css2?family=Bungee&family=Space+Grotesk:wght@400;500;600&family=DM+Mono:wght@400;500&display=swap');

:root {

  /* ── TYPOGRAPHY ── three fonts, three jobs ── */
  --font-brand:              'Bungee', sans-serif;      /* logo, headlines, buttons, nav */
  --font-body:               'Space Grotesk', sans-serif; /* body copy, labels, forms      */
  --font-mono:               'DM Mono', monospace;        /* badges, tags, metadata        */
  --fw-brand-regular:        500;
  --fw-brand-bold:           700;
  --fw-body:                 400;
  --fw-body-medium:          500;
  --fw-mono:                 400;

  /* ── BACKGROUNDS — warm neutral, not pure white ── */
  --color-bg-primary:        #F5F4F0;
  --color-bg-secondary:      #EDECE8;
  --color-bg-tertiary:       #E4E3DF;
  --color-surface:           #FFFFFF;

  /* ── TEXT ── */
  --color-text-primary:      #1A1917;
  --color-text-secondary:    #6B6A65;
  --color-text-tertiary:     #9E9D97;

  /* ── ACCENTS ── */
  --color-lime:              #C8F135;  /* primary action    */
  --color-lime-ink:          #2a4000;  /* text on lime      */
  --color-electric:          #7B5EA7;  /* brand anchor      */
  --color-electric-ink:      #f0e6ff;  /* text on electric  */
  --color-coral:             #FF6147;  /* alert / urgency   */
  --color-sky:               #4DAEEC;  /* info / connect    */
  --color-mint:              #3ECFAC;  /* success           */
  --color-candy:             #FF8DC7;  /* delight / fun     */
  --color-amber:             #FFB830;  /* warning           */

  /* ── PIXEL SYSTEM — no border-radius on pixel components ── */
  --pixel-unit:              8px;
  --pixel-border:            2px solid #1A1917;
  --pixel-shadow:            3px 3px 0 #1A1917;
  --pixel-shadow-lime:       3px 3px 0 #C8F135;
  --pixel-shadow-electric:   3px 3px 0 #7B5EA7;
  --pixel-shadow-coral:      3px 3px 0 #FF6147;

  /* ── BORDER-RADIUS — non-pixel surfaces only ── */
  --radius-sm:               4px;
  --radius-md:               8px;

  /* ── SPACING — 8px pixel grid ── */
  --sp-1:  4px;

  --sp-3:  12px;
  --sp-4:  16px;
  --sp-6:  24px;
  --sp-8:  32px;
  --sp-12: 48px;
  --sp-16: 64px;

  /* ── MOTION ── */

}
